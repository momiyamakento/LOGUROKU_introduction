# LOGUROKU_introduction
Flutter・Firebase を用いて日記を成長マップ化するアプリ「LOGROKU」の、機能概要と担当領域をまとめたポートフォリオ用リポジトリです。

## 📌 概要
LOGROKUは、日記を自動で“すごろく化”し、ユーザーの成長を可視化するアプリです。  
未来ケータイプロジェクト2025としてチーム開発を行いました。

## 🎯 開発目的
- 日記を習慣化しやすくする
- 毎日の行動や気分を“見える化”する
- AIによる簡易分析で自己理解を深める

## 🛠 使用技術（プロジェクト全体）
- Flutter / Dart
- Firebase Authentication / Firestore
- Google Cloud Functions
- Riverpod
- Figma（UI/UX設計）
- Python（分析パート）

## 👤 私が担当した範囲（最重要）
- ○○画面のUI設計
- ○○機能の実装
- Firebase 認証周りのセットアップ
- Firestoreのスキーマ設計
- すごろくマス生成ロジックの一部
- チームのタスク管理 / スプリント計画立案
- ユーザーテスト提案 / 課題抽出

## ✨ 工夫した点 / 技術的チャレンジ
- 状態管理を Riverpod に統合してコードを整理
- Firestore のデータ構造を改善しレスポンスを向上
- UI改善のためユーザーヒアリングを実施
- チーム開発としてWBSを作成し進捗管理を主導
